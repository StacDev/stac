# Delay Action

The `StacDelayAction` allows you to do nothing for a specified duration.

## Delay Action Properties

| Property | Type              | Description                                       |
| --- |-------------------|---------------------------------------------------|
| milliseconds | `int` | The amount of time (in milliseconds) we want to wait for |

## Delay Action JSON

```json
{
    "actionType": "delay",
    "milliseconds": 1000
},

```
